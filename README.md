# 25.11.18.book.001

Custom interactive practice lab for Python data science and AI fundamentals.

## Getting started

1. Install dependencies (Streamlit is required):
   ```bash
   pip install streamlit pandas scikit-learn
   ```
2. Launch the practice lab:
   ```bash
   streamlit run app.py
   ```
3. Open the provided local URL in your browser to work through 20 beginner, 20 intermediate, and 20 advanced exercises. Each exercise includes hints, full solutions, downloadable datasets, and a prompt to confirm when you are ready for the next challenge.
